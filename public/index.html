This is an API-only service. Use the endpoints below to interact with the LiteAPI MCP Server.

GET /api

List all available tools/endpoints

curl https://your-project.vercel.app/api

POST /api

Execute a specific tool

curl -X POST https://your-project.vercel.app/api \
  -H "Content-Type: application/json" \
  -d '{"tool": "tool_name", ...params}'

Note: Make sure to set the LITEAPI_API_KEY environment variable in your Vercel project settings.
